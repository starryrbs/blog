# 设计模式分类

1、创建型模式：创建对象的模式，抽象了实例化的过程。

2、结构型模式：解决怎样组装现有对象，设计交互方式，从而达到实现一定的功能目的。例如，以封装为目的的适配器和桥接，以扩展性为目的的代理、装饰器。

3、行为型模式：描述多个类或对象怎样交互以及怎样分配职责。
